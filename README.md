# Spam Message Detection

This project is a Machine Learning based spam message detection system.
It identifies whether a message is spam or not using NLP techniques.

## Features
- Detects spam and non-spam messages
- Takes user input
- Simple and beginner-friendly project

## Technologies Used
- Python
- Machine Learning
- NLP
- Scikit-learn

## Algorithm
- Naive Bayes Classifier
